# Goals

- High player interaction
- Diverse deckbuilding opportunities
- 4 - 5 players
- Under 2 real hours to play
- Small box

There are 4 basic resource types. each resource can be extracted from its corresponding territory with labor. 

Field = Food
Forest = Wood
Mine = Steel
Quarry = Stone
